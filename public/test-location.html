The API key is being denied. This usually means:

  3. The key doesn't have Geocoding API enabled

To fix:

  1. Go to Google Cloud Console
  2. Find your API key and click "Edit"
  3. Under "Application restrictions":
    • Add http://localhost:3001/* to allowed referrers
    • Or select "None" for testing (not recommended for production)
  4. Under "API restrictions", enable:
    • Places API
    • Geocoding API
